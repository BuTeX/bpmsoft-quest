## 6. Регрессия при добавлении миссии

После каждой новой миссии проверить:

1. новую миссию нельзя открыть раньше времени;
2. старая последняя миссия открывает новую;
3. старое сохранение мигрирует без сброса;
4. XP канонизируется корректно;
5. replay не выдаёт XP повторно;
6. карта и журнал показывают current/complete;
7. desktop и mobile не имеют overflow;
8. все ассеты отвечают HTTP `200`;
9. документация и тестовый expected output обновлены.
